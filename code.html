The Code for HUNDO


//Get the Values from the interface
//Starts our controller function
function getValues() {

    //get values from the page
    let startValue = document.getElementById("startValue").value;
    let endValue = document.getElementById("endValue").value;
    let numbers = [];

    //we need to validate our input
    //Parse into Integers
    startValue = parseInt(startValue);
    endValue = parseInt(endValue);

    if(Number.isInteger(startValue) & Number.isInteger(endValue)) {
        //we call generateNumbers
        numbers = generateNumbers(startValue, endValue);
        //we call displayNumbers
        displayNumbers(numbers);
    }
    else {
        alert("You must enter integers");
    }    

}

//Generate numbers from the startValue to the endValue
//Logic function(s)
function generateNumbers(sValue, eValue){    
        
    let numbers = [];
    
    //we want to get all numbers from start to end.
    //index = index + 1; >>>>> same as index++
    for(let index = sValue; index <= eValue; index++){
        
        //This will execute in a loop until index = endValue.
        numbers.push(index);
    }        
    
    return numbers;    
}

//Display the numbers and mark even numbers bold
//Display or view function(s)
function displayNumbers(numbers){
    
    let templateRows = "";

    for (let index = 0; index < numbers.length; index++) {

        let className ="even";

        let number = numbers[index];

        if(number % 2 == 0){
            className = "even";
        }
        else{
            className = "odd";
        }
        //This does render correctly with Prism see the source code.
        templateRows += `<tr><td class="${className}">${number}</td></tr>

    }

    document.getElementById("results").innerHTML = templateRows;
}

The Code is structured in Three function(s).

Function: getValues

For the first function, this gathers the input from the textbox on the page, using a parsing method, the code validates the input to make sure that what was entered was an integer. The code then runs through a if loop to set the integers, and then shows an alert if the output is incorrect.

Function: generateNumbers

For the second function, the code sets new parameters and sets up an empty array. then runs a for loop to validate that index is equal to sValue, or index is less than or equal to eValue. then adds the numbers by 1 each cycle of the loop. Then adds the numbers to index until it matches the integer in the eValue variable. Which is then sent back to the numbers variable.

Function: displayNumbers

For the third function, the function passes a parameter of numbers to store the value of the count and iterations of generateNumbers. It is then dived by 2 to get even numbers, while the others are odd so that the bold font can be added to distinguish betwween them.
